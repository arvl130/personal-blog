# 📝 Personal Blog

My personal blog that I use to share my thoughts, interests, and learnings.

## Setup

Install the project dependencies:

```sh
pnpm install
```

Start the development server:

```bash
pnpm run dev
```

Build the application for production:

```bash
pnpm run build
```
